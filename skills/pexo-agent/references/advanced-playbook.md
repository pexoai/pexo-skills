# Pexo Agent — Advanced Playbook

## Parallel Creation

Run multiple creative directions simultaneously. Useful for A/B testing concepts or presenting options to stakeholders.

```
"Generate videos for both concepts"
"Take concept A but make it more energetic"
"Show me three different style options"
```

## Storyboard-First Workflow

Have Pexo output a shot-by-shot storyboard before committing to production. Review narrative structure, scene composition, and timing before any rendering begins.

```
"Show me the storyboard first"
"Let me review the script before you start producing"
"Break this into scenes and let me approve each one"
```

## Video Continuation

Extend an existing video by appending new content. No need to regenerate what already works.

```
"Add 10 more seconds at the end with a product showcase"
"Append a closing scene"
"Continue this video with a call-to-action segment"
```

## Multimodal Reference

Supply any asset as a style reference — images, video clips, audio tracks, URLs. No need to decompose aesthetics into parameters.

```
"Match the style and rhythm of this video"
"Use this track as background music"
"Reference the mood and color tone of this image"
"Make something like what's on this product page"
```

## Visual Annotation Feedback

Scrub through video frame by frame and circle areas you want changed. More precise than text descriptions — like pointing at a screen and saying "fix this part."

## Composing with Other Skills

Pexo Agent coordinates VideoAgent studios internally, but it also works as a node in larger agent pipelines:

- **Research → Pexo**: Feed research findings or competitor analysis as creative input
- **Pexo → Distribution**: Chain video output into social publishing or ad platform skills
- **Image gen → Pexo**: Use generated keyframes or concept art as visual references
- **Data → Pexo**: Turn dashboards, reports, or data narratives into motion graphics
